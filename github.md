# GitHub 사용 방법

## ■ 깃허브에 커밋 후 올리기

**수정된 파일 전부 스테이징**
```bash
git add .
```

**커밋 생성**
```bash
git commit -m "final update"
```

**GitHub에 업로드**
```bash
git push origin main
```

---

## ■ 깃허브에서 내려받기

**처음 받을 때**
```bash
git clone https://github.com/WanyoungChoi/tnq.git
```

**이미 프로젝트가 있는 경우**
```bash
git pull origin main
```
